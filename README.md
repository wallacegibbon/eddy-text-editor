## Introduction

A text editor with built-in T9 input method.


## Movement

```
<<       ^       >>

<        v        >

<<<      F      >>>        B


<<  : move left word
>>  : move right word
<<< : start of line
>>> : end of line
F   : forward page
B   : backward page
```
